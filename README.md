# 🛡️ PriceShield

PriceShield is a Flutter and Firebase-based mobile application that helps consumers identify and report product overcharging.

## 🚀 Features

- 📷 Scan product barcode
- 📦 Retrieve product information
- 💰 Compare entered price with the product's MRP
- ⚠️ Detect and highlight overcharging
- 📝 File complaints directly from the app
- 📍 Attach live location or select location from map
- 📸 Upload image/video as proof
- ✍️ Add additional complaint details
- 🔐 Firebase Authentication
- ☁️ Cloud Firestore Database
- 🗂️ Firebase Storage

## 🛠️ Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Google Maps
- Barcode Scanner

## 🎯 Purpose

PriceShield aims to protect consumers from overcharging by providing an easy way to verify product prices and submit digital complaints with evidence.
